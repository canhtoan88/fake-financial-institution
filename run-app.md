# How to run project
1. Clone project
```sh
git clone https://github.com/canhtoan88/fake-financial-institution.git
cd fake-financial-institution
```
2. Installation
```sh
npm i
```
3. Create database (PostgreSQL)
```sql
CREATE DATABASE fake_financial_institution;
```
4. Start app

Development
```sh
npm run start:dev
```
Production
```sh
npm run start:prod
```
